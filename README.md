# Бот для оповещений пользователей

## Запуск

docker compose up

Админка находиться на http://localhost:${FRONTEND_PORT} (по умолчанию http://localhost:9004)

## Настройки

Все конфигурационные перменные должны быть в файле .env (пример в .env.example)
